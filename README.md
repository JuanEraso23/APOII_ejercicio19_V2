# APOII

Curso de Algoritmos y Programación II.

## Descripción

Programa en Java que permite visualizar tablas de multiplicar mediante un menú interactivo.

El usuario puede seleccionar un número del 1 al 10 y el sistema mostrará automáticamente su tabla de multiplicar desde 0 hasta 10.

Características implementadas:
- Uso de ciclos repetitivos (`while` y `for`).
- Estructuras de selección (`switch`).
- Menú interactivo.
- Operaciones matemáticas básicas.
- Opción de salida del programa.

Este ejercicio hace parte de los fundamentos de programación estructurada, estructuras de control y ciclos repetitivos en Java.
